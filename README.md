##  Project Overview
This open-source Python tool is designed specifically for cross-border e-commerce teams (focusing on the Southeast Asian market, e.g., Malaysia/MYR) to automate the analysis of Facebook Ads data. 

By integrating with the OpenAI API, it automatically processes daily ad metrics (ROAS, CPC, CTR), identifies audience fatigue, and generates highly localized livestreaming scripts and ad copies.

##  Key Features
* **Automated Data Cleaning:** Reads Facebook Ads Manager export files (CSV) and normalizes key metrics.
* **LLM-Powered ROAS Analysis:** Sends structured daily performance data to OpenAI to get actionable optimization insights.
* **Localized Content Generation:** Automatically generates A/B testing ad copies tailored for the Malaysian Chinese demographic.
